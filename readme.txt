This is an appliaction which intimates recipient about his friend's birthday by sending him mail on his gmail id.
It will send you mail if any of your friend's birthday matches with current date.

recipient@gmail.com: to whom you want to notify, give your gmail id.
sender@gmail.com: we need one sender to send email. give your gmail id.
password: password of sender.
birthday.xlsx: file where birthdays are stored.    


Note:
1.birthdayMail is an executable jar.
2.birthday.xlsx is a data store for your friend's birthdays.
3.Step to run the appliacation.

java -jar birthdayMail.jar  {recipient@gmail.com} {sender@gmail.com} {password} {birthday.xlsx}

to run this command java should be installed on my machine.
